My research focuses on medical AI, robust visual recognition, and deployment-oriented computer vision systems.

## Research Snapshot

- **Current focus:** Lightweight medical image segmentation, robust preprocessing in difficult conditions, and deployable vision pipelines.
- **Research interests:** Medical AI, computer vision, object detection, and edge / on-device systems.
- **Technical stack:** Python, PyTorch, OpenCV, Linux, and Raspberry Pi / embedded prototyping.
